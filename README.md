# Vue2-Webpack2-Recipe

> A Vue.js and Webpack 2 Recipe

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

```

## Dependencies

- babel-core
- babel-loader
- babel-preset-latest
- cross-env
- css-loader
- file-loader
- vue (2)
- vue-loader
- vue-resource (optional)
- vue-router
- vue-template-compiler
- webpack (2)
- webpack-dev-server
